# ライフプラン・貯蓄計画まとめ(2026年7月版)

> 対象: 上智大学大学院 2029年度入学のための資金準備/二人暮らしでの住宅購入/老後資金
>
> ⚠️ **注意**: マネーフォワード(https://moneyforward.com/bs/portfolio) はログインが必要なためアクセスできませんでした。本プランは公開情報に基づく汎用プランです。「あなたの数字を当てはめる」章の手順で、実際の資産額・月次収支を反映してください。
>
> ⚠️ 本資料は情報整理のためのシミュレーションであり、金融商品の勧誘・専門的な投資助言ではありません。金額はすべて2026年7月時点の公開情報に基づく概算です。

## 全体サマリー: 3つのゴールと必要額

| ゴール | 期限 | 目標額(概算) | 月あたり積立の目安 |
|---|---|---|---|
| ① 上智大学大学院(修士2年) | 2029年3月まで | **約200万円**(文系) / 約250万円(理系) | 約6.5万円/月(残り32か月) |
| ② 住宅購入の頭金+諸費用 | 修了後5年(〜2036年頃) | **約1,000〜1,500万円** | 約17〜25万円/月(二人合算・5年) |
| ③ 老後資金(65歳時点) | 〜65歳 | **約1,800〜2,500万円** | 月3〜5万円の積立投資(30年・年4%想定) |
| (常時) 生活防衛資金 | 常に確保 | 生活費6か月分(例: 180万円) | 最優先で確保 |

3つを同時に満額は難しいため、**時期をずらしたフェーズ制**で進めるのが現実的です(後述)。

## ① 上智大学大学院 2029年度入学の資金計画

### 学費(2026年度・博士前期課程/修士課程の公表額ベース)

文科系研究科(心理学・看護学専攻除く)の例:

| 項目 | 金額 |
|---|---|
| 入学金(入学時のみ) | 200,000円 |
| 在籍料 | 30,000円/学期(年60,000円) |
| 授業料 | 252,000円/学期(年504,000円) |
| 教育充実費 | 90,000円/学期(年180,000円) |
| 学生保険料(入学時) | 1,400円 |
| 同窓会費(最終年次) | 40,000円 |
| **初年度合計** | **約945,400円**(うち入学手続時=春学期分 約573,400円) |
| **2年目合計** | **約784,000円** |
| **修士2年間の総額** | **約173万円** |

**心理学専攻(臨床心理学コース含む)は上記の標準額の対象外**で、授業料が高く設定されています(授業料 約295,500円/学期)。概算は以下の通り:

| 項目 | 金額(心理学専攻・概算) |
|---|---|
| 入学手続時(入学金+春学期分) | 約617,000円 |
| 初年度合計 | 約1,032,000円 |
| 2年目合計 | 約871,000円 |
| **修士2年間の総額** | **約190万円**(+臨床実習関連費の可能性あり、要公式確認) |

- 理工学研究科など理系は実験実習研究費等が加わり、**2年総額で約200〜250万円**を見込むと安全です。
- 2029年度入学時には学費改定の可能性があるため、**+5%程度のバッファ**を推奨。
- 出典: [上智大学 大学院学費一覧](https://piloti.sophia.ac.jp/jpn/tuition/daigakuingakuhi/)、[2026年度学費PDF](https://piloti.sophia.ac.jp/assets/uploads/2025/12/38da8a5e54760f75ec4b41ee44326448.pdf)、[入試情報(学費・奨学金)](https://adm.sophia.ac.jp/jpn/in_ad/graduate_gakuhi/)。※2029年度入学向けの最新額は必ず公式で確認してください。

### 学費以外に必要なお金

| 項目 | 概算 |
|---|---|
| 受験料(検定料) | 約3.5万円/回(併願なら×回数) |
| 書籍・PC・学会費など | 年10〜20万円 |
| 通学交通費 | 定期代(四谷キャンパス想定で実費) |

### 積立プラン(今から入学まで)

- 今(2026年7月)→ 2029年3月まで **残り32か月**。
- 目標200万円 ÷ 32か月 = **約6.3万円/月**。理系想定(250万円)なら**約7.8万円/月**。
- **入学手続時(合格後、2028年秋〜2029年2月頃)に初年度春学期分 約57〜70万円をまとめて払う**ため、2028年秋までに最低100万円は現金で確保しておく。
- 使う時期が3年以内のお金なので、**投資には回さず、普通預金・定期預金・個人向け国債(変動10年)など元本確保型**で貯める。

### 負担を減らす制度(必ず検討)

| 制度 | 内容 |
|---|---|
| JASSO 第一種奨学金(無利子) | 修士: 月5万円 or 8.8万円。**特に優れた業績による返還免除**(全額/半額)あり |
| JASSO 第二種奨学金(有利子) | 月5〜15万円 |
| 上智大学の給付型奨学金・特待生制度 | 学内選考。入試成績優秀者対象のものも |
| TA/RA(ティーチング/リサーチアシスタント) | 収入+研究実績 |
| 専門実践教育訓練給付金 | 社会人で雇用保険加入歴があり、対象講座指定されている研究科なら**受講費の最大50〜80%(年間上限あり)**が戻る。厚労省の[講座検索システム](https://www.kyufu.mhlw.go.jp/kensaku/)で対象か要確認 |

## ② 二人暮らし・住宅購入のファイナンシャルプラン

### 前提となる市場環境(2026年時点)

- 首都圏の新築マンション平均価格は**2025年度で9,383万円**と過去最高水準([日経新聞](https://www.nikkei.com/article/DGXZQOUC204F80Q6A420C2000000/))。東京23区の新築は1億円超が常態化。
- 首都圏の**中古マンション成約価格は平均約5,000万円台**(2026年5月、[東京カンテイ](https://www.kantei.ne.jp/report/)等)。都心3区では調整の兆しも。
- 住宅ローン金利は上昇局面: **変動 約1.0〜1.5%、フラット35 約3.1%**(2026年7月時点、[モゲチェック](https://mogecheck.jp/mortgage-ranking/purchase/fixed-35)、[住宅金融支援機構](https://www.simulation.jhf.go.jp/flat35/kinri/index.php/rates/top))。

### 現実的なターゲット設定(例)

二人暮らし(50〜70㎡)で、**物件価格5,000〜7,000万円**(郊外新築 or 都内中古)を想定。以下は**6,000万円の物件**での試算です。

| 項目 | 金額 |
|---|---|
| 物件価格 | 6,000万円 |
| 頭金(10〜15%) | 600〜900万円 |
| 諸費用(仲介手数料・登記・ローン費用等、中古は6〜10%) | 300〜500万円 |
| **購入時に必要な現金** | **約1,000〜1,400万円** |
| 借入額(頭金900万円の場合) | 5,100〜5,400万円 |

### 月々の返済シミュレーション(借入5,400万円・35年・元利均等)

| 金利 | 月々返済 | 総返済額 |
|---|---|---|
| 変動 1.3% | **約16.0万円** | 約6,720万円 |
| 固定 3.1%(フラット35) | **約21.1万円** | 約8,860万円 |

- 無理のない返済負担率は**手取り月収の20〜25%以内**が目安。月16万円の返済なら**世帯手取り月64〜80万円(世帯年収約1,000万円前後)**が安全圏。届かない場合は物件価格を下げるか頭金を厚くする。
- 二人で買うなら**ペアローン or 収入合算**で借入枠を確保できるが、どちらかの収入減(大学院進学・産休等)に耐えられる返済額に抑えるのが鉄則。
- **住宅ローン控除**(年末残高の0.7%を最大13年間税額控除、省エネ基準等の要件あり)を織り込むと、当初は実質金利がさらに下がる。

### 購入時期と積立プラン

- 大学院在学中(2029〜2031年)は貯蓄が細るため、**購入は修了・復職後の2032〜2036年頃**が現実的。
- 目標1,200万円 ÷ 5年(60か月) = **月20万円(二人合算・ボーナス込み)**。共働きで世帯手取り45〜55万円/月なら貯蓄率35〜45%で達成可能な水準ですが、厳しければ「7年で月14万円」に伸ばす。
- 頭金づくりも5年以内に使うお金なので**元本確保型が基本**。ただし7年以上先なら一部(2〜3割)をNISAで運用する選択肢も。

## ③ 老後資金プラン

### 年金と支出のギャップ(2026年度の公表値ベース)

| 項目 | 月額 |
|---|---|
| 夫婦2人の標準的な厚生年金受給額(モデル世帯) | 約23.7万円 |
| 高齢夫婦無職世帯の平均支出(消費+税・社保) | 約28.7万円 |
| **不足額** | **約5.0万円/月** |

- 65〜95歳の30年間で不足は **約1,800万円**。これがいわゆる「老後2,000万円」の実体です([総務省家計調査ベース](https://www.resonabank.co.jp/kojin/column/taishoku_unyo/column_0003.html)、[マネコミ](https://manekomi.tmn-anshin.co.jp/kakei/17645387))。
- **共働きで二人とも厚生年金**なら受給額は月28〜32万円程度まで増え、不足はかなり圧縮されます。逆に「ゆとりある老後」(旅行・趣味込みで月約38万円)を目指すなら**不足は月10〜14万円 → 4,000〜5,000万円**規模。
- まずは**ねんきん定期便 or [ねんきんネット](https://www.nenkin.go.jp/n_net/)で二人の見込額を確認**するのが第一歩。

### 積立投資でいくら作れるか(年利4%想定・複利)

| 毎月の積立額 | 20年後 | 25年後 | 30年後 |
|---|---|---|---|
| 3万円 | 約1,100万円 | 約1,540万円 | **約2,080万円** |
| 5万円 | 約1,830万円 | 約2,570万円 | **約3,470万円** |
| 7万円 | 約2,570万円 | 約3,600万円 | 約4,860万円 |

→ **35歳前後から月3〜5万円をNISAで積立**できれば、標準的な不足額(1,800〜2,500万円)はカバーできる計算。

### 使う制度の優先順位

1. **新NISA**: つみたて投資枠 年120万円+成長投資枠 年240万円、生涯投資枠1,800万円/人(夫婦で3,600万円)。非課税・いつでも引き出せるので最優先。
2. **iDeCo**: 掛金が全額所得控除(会社員は月2.3万円〜。拠出上限の引き上げが決定済みのため最新の限度額を要確認)。60歳まで引き出せないので「老後専用」と割り切れる分だけ。
3. **退職金・企業型DC**: 勤務先の制度を確認し、マッチング拠出があれば活用。

## フェーズ別・実行プラン(全体の時間軸)

```
2026 ──── 2029 ──────── 2031 ─────────── 2036 ──────────── 65歳
  │ Phase1  │   Phase2    │     Phase3      │     Phase4
  │ 院試資金 │  大学院在学  │  住宅頭金づくり   │  老後資金の本格積立
```

| フェーズ | 期間 | やること | 積立の目安 |
|---|---|---|---|
| **Phase 1** | 2026.7〜2029.3 | 生活防衛資金6か月分を確保 → 大学院資金200万円を預金で積立。NISAは少額(月1〜2万円)で継続し習慣化 | 貯蓄 約6.5万円/月 + NISA 1〜2万円/月 |
| **Phase 2** | 2029.4〜2031.3 | 大学院在学。学費支払い期。奨学金・TA/RA・給付金で持ち出しを最小化。パートナーの収入で家計維持、貯蓄は無理しない | NISA月1万円は死守 |
| **Phase 3** | 2031〜2036頃 | 復職・収入回復。二人で住宅頭金+諸費用1,200万円を集中積立 → 購入 | 二人で15〜20万円/月 |
| **Phase 4** | 購入後〜65歳 | ローン返済と並行して老後資金へシフト。NISA/iDeCoで月3〜5万円以上 | 投資 3〜5万円/月〜 |

**優先順位の原則**: 生活防衛資金 > 期限が近い大学院資金(元本確保) > 住宅頭金(元本確保中心) > 老後資金(長期投資)。ただし老後資金は「時間」が最大の武器なので、少額でも早く始めて止めないこと。

## あなたの数字を当てはめる(マネーフォワード活用)

本プランを実データで精緻化する手順:

1. [マネーフォワードのポートフォリオ画面](https://moneyforward.com/bs/portfolio)で以下をメモ:
   - **現預金合計** = A円/**投資資産(株・投信等)合計** = B円/**負債(ローン・カード残)** = C円
2. 家計簿画面で直近6か月の**平均月次収支(黒字額)** = D円/月 を確認。
3. 当てはめ:
   - 生活防衛資金の充足: A −(月支出×6)= 余裕額
   - 大学院資金の残り必要額: 200万円 − (Aの余裕額のうち充当可能分) → ÷32か月で毎月の積立額を再計算
   - D円/月 が「Phase1の必要積立(約7〜8万円)」を下回るなら、家計簿の**固定費(通信・保険・サブスク)から削減**するのが定石
4. マネーフォワードの**「予算」機能**で上記の毎月積立額を予算化し、**「資産推移」**で月次モニタリング。目的別に口座を分ける(教育資金用の定期預金口座など)と管理が楽です。
