# BOSCO NEXT サイト 編集ガイド

**ふだんの更新は、ほぼ全部「編集画面（/admin）」のフォームでできます。** HTMLやGitHubを触る必要はありません。

---

## A. 編集画面（/admin）でできること ★ほぼ全部こちら

`あなたのサイトURL/admin`（例：`https://bosconext-futsal.netlify.app/admin/`）を開く → ログイン → 左のメニューから項目を選ぶ → フォームを入力 → **「Publish（公開）」** → 1分ほどでサイトに反映。

編集できる項目（左メニュー）:
- **お知らせ** … 大会結果・活動報告・お知らせ。上にあるものほど新着表示。
- **トップの文章** … キャッチコピー、紹介文、成績バッジ、テロップ、メイン写真 など。
- **チーム・選手** … チーム紹介・チームデータ・選手（背番号/名前/写真）・スタッフ。
- **スケジュール** … これからの予定／これまでの結果。**予定の一番上が「次の一戦」**として大きく出ます（トップの「次の一戦」もここと連動）。
- **大会戦績** … 大会名と結果（優勝／準優勝／出場）。
- **フォトギャラリー** … 写真を**その場でアップロード**して追加。
- **リンク** … SNSや関連団体のリンク。

### 入力のコツ
- **写真**は、選手・ギャラリー・トップのメイン写真とも、フォームでアップロードできます。
- **キャッチコピー**は `{ }` で囲んだ言葉がゴールドで強調されます（例：`どんな壁も、{乗り越えられる}。`）。
- バッジやテロップ、紹介文の段落は「＋」で1行ずつ追加できます。
- 並べ替えはドラッグ。削除も各項目から。

### 編集する人を増やす
**DecapBridge** のダッシュボードで、編集者を**メール招待**します（招待された人はGitHubアカウント不要でログインできます）。

## B. デザインや一部のページを直すとき（GitHub）

次のものは、まだ編集画面に入れていないので、GitHubでファイルを編集します（ファイルを開く →鉛筆アイコン → 直す →「Commit changes」→ 自動公開）。

- **色・全体のデザイン** … `style.css`（上部の `:root` のカラーコード）
- **メニュー・SNSリンク・チーム名** … `site.js` の `SITE_INFO`
- **リーグ** … `league.html`／**新聞・雑誌切り抜き** … `press.html`／**会員専用** … `members.html`
- **お問い合わせフォームの通知先メール** … Netlifyの「Forms」設定（チームのGmール）

> これらも編集画面（フォーム）に追加できます。必要になったら教えてください。

---

## C. 仕組み（参考）

- 編集画面で入力した内容は、`〇〇.json`（news.json／home.json／team.json…）というデータファイルに保存され、各ページがそれを読み込んで表示します。
- パソコンでHTMLファイルを直接ひらいた“プレビュー”では、各ページに内蔵した予備データが表示されます（ネット上では .json が読まれるので問題ありません）。

## D. 困ったとき
- 反映されない → 1〜2分待つ／ブラウザを再読み込み。
- 間違えた・消した → GitHubの「履歴（Commits）」から元に戻せます。
- 直したい箇所を伝えてもらえれば、こちらで対応します。
